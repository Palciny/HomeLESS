The objective of this project is a cross-platform free laser shooting simulator available to the average user. This solution is using ordinary webcam and ordinary laser pointer. HomeLESS is a free software. Anybody can download the source code and improve it oneself. HomeLESS is developed in open source programming language and environment Processing and can be used in both GNU/Linux and Microsoft Windows.
